Bem vindo!

NPortugol é uma biblioteca de linguagem estruturada baseada numa variação da sintaxe popularmente conhecida como Portugol (utilizada para representação e ensino de algoritmos sem o rigor de uma linguagem formal).

Para um começo rápido, leia o Tutorial da Linguagem para conhecer as regras e sintaxes. Você pode editar e executar programas com a ferramenta TurboNPortugol.

Para interessados no processo de compilação e execução, utilize a ferramenta GrammarIDE para executar programas, visualizar AST gerada no parser e o bytecode gerado para a máquina virtual.

Usando NPortugol como Scripting Language

O NPortugol pode ser utilizado como scripting language hospedando o runtime em qualquer aplicação .NET, aproveitando o recurso de interoperabilidade. Isso possibilita, por exemplo, ter a linguagem como base para uma external DSL. Veja como no Tutorial Scripting

Exemplo C# => NPortugol

var engine = new Engine();

engine.Compile("funcao soma(a, b) retorne a + b fim");

var result = engine.Execute("soma", 2, 2);

Exemplo NPortugol => C#

var engine = new Engine();

engine.HostContainer.Register("hoje", x => DateTime.Now.ToShortDateString());

engine.Compile(@"funcao principal() 
                        variavel data
                        data = hoje()
                 fim");

engine.Execute();
